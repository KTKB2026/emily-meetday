# Emily meet day

Public meet-day card for Emily (MASS Strength, Olympic weightlifting).
Single `index.html` plus `sw.js` (offline cache), served via GitHub Pages
from github.com/KTKB2026/emily-meetday.

Public repo rules: first name only, no contact info, only numbers that are
already public meet data (attempts, PRs, entry total).

Bump the `CACHE` name in `sw.js` on every deploy so phones pick up changes.

This folder is canonical. Global rules: `~/.claude/CLAUDE.md`.
